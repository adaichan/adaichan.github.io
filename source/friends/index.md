# 友链交换

想交换友链的小伙伴，欢迎在评论区留言哦，留言格式：

* **名称：**你的博客名称
* **地址：**你的博客地址
* **简介：**一句话简介
* **头像：**你的头像地址

* **名称：**阿呆是也!

* **头像：**http://adaichan.com/medias/avatars/avatar.jpg
